# 画面確認チェックリスト

WRT-helpr の通話中判定タブで、以下のケースを実入力または `docs/demo_cases.md` のコピー文取り込みで確認する。
重点確認対象は `overall_status`、保証期間判定、概算費用表示、判定診断パネル、修理拠点判定の扱い。

## 1. 保証期間終了

### 入力条件
- 製品: 洗濯機
- メーカー: パナソニック
- 都道府県: 滋賀県
- 家電/住設: 家電
- 保証開始日: 2020/01/01
- 保証終了日: 2026/04/26

### 期待結果
- overall_status: `error`
- 保証期間判定: `保証期間終了`、受付不可を最優先表示
- 修理形態: `出張修理`。ただし補助表示扱い
- 概算費用: 大きく案内しない。`保証期間終了後のため、WRT概算案内対象外`
- 参照スクリプト: `家電出張・持込・新築住設` / `家電・出張修理`
- 修理拠点判定: 受付不可が優先。手配対象外または補助表示
- 判定診断パネルで最上位に出るべき項目: `保証期間判定` / `受付不可` / impact=`blocking`
- NG表示になってはいけない項目: 概算費用を通常の緑色確定表示で強く案内しない

### 目視確認チェック
- [ ] 画面上部に保証期間終了の赤警告が出ている
- [ ] `判定診断：受付不可 / 重大確認あり` が大きく表示される
- [ ] 判定診断パネルの先頭が保証期間判定になっている
- [ ] 概算費用より受付不可理由が強く見える
- [ ] 修理拠点は手配対象外または補助表示になっている

## 2. 保証開始日前

### 入力条件
- 製品: エアコン
- メーカー: ダイキン
- 都道府県: 東京都
- 家電/住設: 家電
- 保証開始日: 2026/05/01
- 保証終了日: 2031/04/30
- 補足条件: 空欄

### 期待結果
- overall_status: `warning`
- 保証期間判定: `保証開始日前`、メーカー保証または販売店・メーカー窓口へ誘導
- 修理形態: `出張修理`。ただし補助表示扱い
- 概算費用: WRT概算案内対象外。裏側判定ではダイキンの家庭用/業務用確認が残る
- 参照スクリプト: `家電出張・持込・新築住設` / `家電・出張修理`
- 修理拠点判定: 終話後確認または補助表示
- 判定診断パネルで最上位に出るべき項目: `保証期間判定` / impact=`call_time_required`
- NG表示になってはいけない項目: 概算費用を確定金額として大きく案内しない

### 目視確認チェック
- [ ] 画面上部に保証開始日前の黄色警告が出ている
- [ ] メーカー保証または販売店・メーカー窓口への誘導が見える
- [ ] 判定診断パネルの先頭が保証期間判定になっている
- [ ] 概算費用は未確定または対象外として扱われる
- [ ] 修理形態は出張修理でも、受付可否より弱く表示される

## 3. エアコン + メーカー未入力

### 入力条件
- 製品: エアコン
- メーカー: 空欄
- 都道府県: 東京都
- 家電/住設: 家電
- 保証開始日: 2026/01/01
- 保証終了日: 2030/12/31

### 期待結果
- overall_status: `warning`
- 保証期間判定: `保証期間内`
- 修理形態: `出張修理`
- 概算費用: `未確定`。金額表示なし
- 参照スクリプト: `家電出張・持込・新築住設` / `家電・出張修理`
- 修理拠点判定: 通常どおり、または終話後確認
- 判定診断パネルで最上位に出るべき項目: `概算費用判定` / `通話中確認`
- NG表示になってはいけない項目: `5,000円～7,000円前後` などの汎用出張費用を表示しない

### 目視確認チェック
- [ ] `判定診断：要確認あり` が表示される
- [ ] 概算費用欄が `未確定` または追加確認扱いになる
- [ ] `次に確認：メーカーを確認してください` が目立つ
- [ ] 汎用出張費用が表示されていない
- [ ] 修理形態は `出張修理` として確認できる

## 4. 給湯器のみ

### 入力条件
- 製品: 給湯器
- メーカー: パナソニック
- 都道府県: 東京都
- 家電/住設: 住設
- 保証開始日: 2026/01/01
- 保証終了日: 2030/12/31

### 期待結果
- overall_status: `warning`
- 保証期間判定: `保証期間内`
- 修理形態: `出張修理`
- 概算費用: `未確定`
- 参照スクリプト: `住設【既築／中古のみ】` / `住設受付`
- 修理拠点判定: 通常どおり、または終話後確認
- 判定診断パネルで最上位に出るべき項目: `概算費用判定` / `通話中確認`
- NG表示になってはいけない項目: ガス/石油/ハイブリッドの別が未確認のまま金額を確定表示しない

### 目視確認チェック
- [ ] `判定診断：要確認あり` が表示される
- [ ] 概算費用は `未確定`
- [ ] `次に確認：ガス給湯器・石油給湯器・ハイブリッド給湯器の別を確認してください` が見える
- [ ] 汎用の確定金額に落ちていない
- [ ] 保証期間内であることは緑表示または受付判定へ進める表示になっている

## 5. 正常系: ドライヤーまたは洗濯機

### 入力条件
- 製品: ドライヤー
- 原文シリーズ: ドライヤー・ヘアアイロン
- メーカー: パナソニック
- 都道府県: 東京都
- 家電/住設: 家電
- 保証開始日: 2026/01/01
- 保証終了日: 2030/12/31

### 期待結果
- overall_status: `ok`
- 保証期間判定: `保証期間内`
- 修理形態: `持込修理`
- 概算費用: `2,000円～5,000円前後`
- 参照スクリプト: `家電出張・持込・新築住設` / `家電・持込修理`
- 修理拠点判定: 担当確認が必要な場合は `終話後確認` / impact=`after_call_ok`
- 判定診断パネルで最上位に出るべき項目: 修理拠点が未確定なら `修理拠点判定` / `終話後確認`。全体は `ok` のまま
- NG表示になってはいけない項目: 修理拠点の終話後確認だけで `overall_status=warning` にしない

### 目視確認チェック
- [ ] `判定診断：OK` が表示される
- [ ] 保証期間内として受付判定へ進める
- [ ] 概算費用が `2,000円～5,000円前後` と表示される
- [ ] 修理拠点が終話後確認でも全体は OK のまま
- [ ] 判定診断パネルで `終話後確認` ラベルが確認できる
